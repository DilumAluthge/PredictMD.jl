PredictMD

PredictMD is a free and open-source Julia package that provides a uniform interface for machine learning.

Documentation GitHub

Installation

PredictMD requires Julia version 1.1 or greater.

macOS and GNU/Linux:

Open a new terminal and run the following line:

curl https://predictmd.net/install.jl | julia

Windows and all other platforms
Step 1:
Right-click on the following button, click "Save Link As...", and save the file as install.jl.
Step 2:
Navigate to the location of the file saved in step 1, and run the file with Julia, for example: julia install.jl
